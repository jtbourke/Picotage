# Picotage

PICO-8 game based on Sabotage (1981, Apple II)

**[Play in browser](https://jtbourke.github.io/Picotage/)**

## Run Locally

Open `docs/index.html` in a browser, or start a local server:

```bash
# Python
cd docs && python -m http.server 8000

# Node
npx serve docs
```

Then open http://localhost:8000

## Controls

- Arrow keys: Aim turret
- Z/X: Fire
